For reference, I use the following environment variables:

# Node Environment
NODE_ENV=development
PORT=8000

# Database
DATABASE_URL=postgres://postgres:....

# Authentication
ACCESS_TOKEN_SECRET=your_super_secret_jwt_key_change_this
FRONTEND_URL=http://localhost:3000
CORS_ORIGIN=http://localhost:3000

# Email Configuration (Nodemailer)
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-name@gmail.com
SMTP_PASS=<app-password>
SMTP_SECURE=false

# Sender Identity
FROM_EMAIL=no-reply@gmail.com
FROM_NAME="Sayan Enterprises"

# Stripe Configuration
STRIPE_SECRET_KEY=sk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...
STRIPE_PRO_PRICE_ID=price_...


Modify the file src/config/env.ts accordingly.
